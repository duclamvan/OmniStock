- ‚ÄúMark as Delivered‚Äù button:
    - Ask Amir to design a New Pop-up ‚ÄúImport Preview‚Äù:
        
        ![472590244_1343454486552820_3113533272452368516_n.jpg](https://prod-files-secure.s3.us-west-2.amazonaws.com/6ae63053-f667-4c96-9240-a4a38f787e8e/a31e99d3-7813-453a-8279-d82fe1408089/472590244_1343454486552820_3113533272452368516_n.jpg)
        
    - A table with 4 columns:
        - Item Name, Purchase cost, QtyProp, PriceProp
        - Purchase cost: is **Import Price** from each product from Order List (Purchase List)
        - **QtyProp:**
            - (the algorithm provided below is designed to calculate **landed costs per unit** for a given quantity of items.)
            - Calculation:
            
            ```jsx
            const costs = tmp_costs.map(Number);
            const quantities = tmp_quantities.map(Number);
            
            // Assuming shipping_cost is defined elsewhere
            
            // Calculate total quantity
            var total_quantity = 0;
            for (var i = 0; i < quantities.length; i++) {
              total_quantity += quantities[i];
            }
            
            // Calculate cost per unit from shipping cost
            var cost_per_unit = shipping_cost / total_quantity;
            
            // Calculate landed costs and round to three decimal places
            var landed_costs_q = new Array(costs.length);
            for (var i = 0; i < costs.length; i++) {
              // Calculate the final landed cost per unit, including the shipping cost
              var final_cost_per_unit = costs[i] + cost_per_unit;
              // Round the final cost to 3 decimal places
              landed_costs_q[i] = Math.round(final_cost_per_unit * 1000) / 1000;
            }
            ```
            
            ### `Scenario:`
            
            - `You have the following items:`
                - **`5 pcs of Product A** at a unit cost of **2 USD**`
                - **`3 pcs of Product B** at a unit cost of **3 USD**`
            - `Shipping cost: **4 USD**`
            
            ### `Calculation:`
            
            1. **`Total Quantity**:`
                - `5+3=85 + 3 = 85+3=8 pcs.`
            2. **`Cost Per Unit of Shipping**:`
                - `Shipping¬†Cost¬†Per¬†Unit=48=0.50\text{Shipping Cost Per Unit} = \frac{4}{8} = 0.50Shipping¬†Cost¬†Per¬†Unit=84=0.50 USD per unit.`
            3. **`Landed Cost Per Unit**:`
                - `Product A: 2+0.50=2.50 USD`
                    
                    `2+0.50=2.502 + 0.50 = 2.50`
                    
                - `Product B: 3+0.50=3.50 USD`
                    
                    `3+0.50=3.503 + 0.50 = 3.50`
                    
            4. **`Results**:`
                - `Landed Costs (Rounded to 2 decimals):`
                    - `Product A: **2.50 USD**`
                    - `Product B: **3.50 USD**`
        - **PriceProp:**
            - this algorithm is specifically designed to calculate **product import costs** using a **proportional cost allocation**
            - Calculation:
            
            ```jsx
            const costs = tmp_costs.map(Number);
            const quantities = tmp_quantities.map(Number);
            
            // Calculate the total value of all items (considering the quantities)
            var total_value = 0;
            for (var i = 0; i < costs.length; i++) {
              total_value += costs[i] * quantities[i];
            }
            
            // Calculate the cost proportion for each item based on the total value
            var cost_proportions = new Array(costs.length); // Pre-allocate array with fixed size
            for (var i = 0; i < costs.length; i++) {
              cost_proportions[i] = (costs[i] * quantities[i]) / total_value;
            }
            
            // Calculate the final unit costs including their share of the shipping cost
            var landed_costs_price = new Array(costs.length); // Corrected array size
            for (var i = 0; i < costs.length; i++) {
              var cost_proportion = cost_proportions[i];
              var additional_cost = cost_proportion * shipping_cost;
              // Distribute the additional cost per unit based on the quantities
              var final_unit_cost = costs[i] + (additional_cost / quantities[i]);
              landed_costs_price[i] = Math.round(final_unit_cost * 1000) / 1000; // Round to 3 decimal places
            }
            ```
            
            ### **`Scenario:`**
            
            - `You have the following items:`
                - `5 pcs of Product A at a unit cost of 2 USD`
                - `3 pcs of Product B at a unit cost of 3 USD`
            - `Shipping cost: 8 USD`
            
            ### **`Calculation:`**
            
            1. `Total Value:`
                - `Product A: 5 √ó 2 = 10 USD`
                - `Product B: 3 √ó 3 = 9 USD`
                - `Total Value = 10 + 9 = 19 USD`
            2. `Cost Proportion:`
                - `Product A: 10 / 19 = 0.5263 (52.63%)`
                - `Product B: 9 / 19 = 0.4737 (47.37%)`
            3. `Shipping Cost Allocation:`
                - `Product A: 0.5263 √ó 8 = 4.21 USD`
                - `Product B: 0.4737 √ó 8 = 3.79 USD`
            4. `Landed Cost Per Unit:`
                - `Product A: 2 + (4.21 / 5) = 2 + 0.84 = 2.84 USD`
                - `Product B: 3 + (3.79 / 3) = 3 + 1.26 = 4.26 USD`
            
            `Results:`
            
            - `Landed Costs:`
                - `Product A: 2.84 USD`
                - `Product B: 4.26 USD`
